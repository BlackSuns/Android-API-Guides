# 概述 #

+ Fragment将应用程序的功能和UI分解成可重用的模块。
+ 添加多个Fragment来避免Activity之间的切换。
+ Fragment拥有自身的生命周期、状态和回退栈。
+ Fragment要求API 11以上（当然也可以用v4支持包）。
